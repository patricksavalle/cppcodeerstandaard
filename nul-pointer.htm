Regel: nul-pointer

Gebruik 0 om de nul-pointer mee aan te geven, niet NULL.

NULL is in C++ toch al gedefinieerd als int(0) en niet zoals in C als (void*)0. De laatste kan namelijk niet worden toegekend aan pointers van een ander type zonder typecast vanwege de strong-typing van C++! Het onderstaande programma zal voor de meeste compilers (niet GNU) melden dat het type van NULL inderdaad een int is: 

  #include 

  void Type(int)
  {
    cout << "Is number.\n";
  }

  void Type(void*)
  {
    cout << "Is pointer.\n";
  }

  int main()
  {
    Type(NULL);
  }
