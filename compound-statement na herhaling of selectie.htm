Regel: compound-statement na herhaling of selectie

Elk herhalings- of selectie-statement wordt gevolgd door een (eventueel leeg) compound-statement.

Een compound-statement is een blok met statements: 

  // compound-statement
  {
    // bla bla
  }

Deze regel geldt voor de if, else, while, do, en de for. 

  // niet zo
  {
    if ( expressie ) statement;

    if ( expressie )
      statement;
    else
      statement;

    while( expressie ) statement;

    do statement; while( expressie );

    for( int i=0; i<10; i++) statement
  }

  // maar zo
  {
    if ( expressie ) 
    {
      statement; 
    }

    if ( expressie )
    {
      statement; 
    }
    else
    {
      statement;
    }

    while( expressie ) 
    {
      statement;
    }

    do
    { 
      statement; 
    }
    while( expressie );

    for( int i=0; i<10; i++) 
    {
      statement;
    }
  }

Voorzie ook een leeg herhalingsstatement voor de duidelijkheid van een leeg compound-statement. Soms komt het voor dat een programmeur ervoor kiest om een 'lege' lus te programmeren. Bijvoorbeeld: 

  {
    const int WACHT_TIJD=10;    
    time_t time_mark = time();
    while (time() < (time_mark + WACHT_TIJD));
  }

Duidelijker is echter om deze constructie als volgt te coderen: 

  {
    const int WACHT_TIJD=10;
    time_t time_mark = time();
    while (time() < time_mark+WACHT_TIJD) 
    {
      // niets
    }
  }
