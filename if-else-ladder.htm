Regel: if-else-ladder

Het laatste statement van een if-else-ladder is een else-block.

Als een if-else-ladder geen afsluitende else heeft, laat dit dan expliciet zien en noteer een leeg block. De volgende die de code leest weet dan dat het ontbreken van de else geen vergissing was! 

  // onduidelijk 
  {
   if(a<0)
    {
      // iets 
    }
    else if (b<0)
    {
      // iets anders 
    }
  }

  // duidelijker 
  {
   if(a<0)
    {
      // iets 
    }
    else if (b<0)
    {
      // iets anders 
    }
    else
    {}
  }
