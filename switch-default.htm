Regel: switch-default

Elk switch-statement moet een default hebben.

Het default-label is de ideale plek om het waardebereik van de te testen variabele te valideren. De volgende constructie zal dus regelmatig (moeten) voorkomen: 

  {
    switch(a)
    {
      case 1:
        // bla bla 
        break;

      case 2:
        // bla bla 
        break;

      default:
        throw Exception("ongeldig waardebereik . . .");
        break;
    }
  }
