Codeerstandaard voor C++

Regel: class-section-volgorde

Elke class declareert z'n public-, protected- en private-sectie bij voorkeur in deze volgorde.

Een voorbeeld: 

  class SomeClass
  {
  public:
    SomeClass();
    SomeClass( const SomeClass& copy );
    SomeClass& operator=( const SomeClass& copy );
    virtual ~SomeClass();
  protected:
    int protected_member();
  private:
    int private_member();
  };
